# Registro do uso de IA

## Ferramenta de IA utilizada

- ChatGPT para apoiar a definicao dos cenarios e o refinamento dos seletores.

## Exemplos de prompts utilizados

1. "Gere dois cenarios E2E em Playwright para um site publico com cadastro, login e navegacao."
2. "Sugira validacoes robustas para o fluxo de cadastro do site Automation Exercise."
3. "Crie alternativas de seletores para um botao de signup usando estrategias por role, placeholder e CSS."
4. "Como implementar fallbacks de localizadores em Playwright para reduzir fragilidade dos testes?"

## Como documentar no relatorio

- Informar que a IA foi usada como suporte na escrita do codigo.
- Explicar que todo o codigo foi revisado e validado manualmente.
- Destacar que a IA acelerou a criacao dos testes, mas nao substituiu a analise critica do aluno.
